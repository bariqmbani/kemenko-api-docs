Dokumen

Bagian ini berisi hal yang diperlukan untuk handle dokumen, seperti menyimpan data, mengambil data, dan menghapus data. Query parameter type perlu dikirimkan pada API dokumen ini karena untuk me-handle dokumen Laporan Monev atau dokumen Program GNRM.

query
type *
  • monev
  • gnrm
    • Status Code
    400
    Query parameter type tidak sama dengan 'monev' atau 'gnrm'. 
                                
                                {
                                    "type": "error",
                                    "message": "Request tidak valid"
                                }

    Mendapatkan daftar dokumen

    GET
    /api/document
    headers
    x-auth-token *
    aweuaweu <token>
    query
    instansi
    <nama instansi> (cth: Kemenko PMK)
    Mendapatakan dokumen dari instansi terkait (hanya bisa oleh 'owner')
    limit
    <angka> (cth: 10)
    Batasan total data yang akan diterima dan ditampilkan
    page
    <angka> (cth: 2)
    Menampilkan halaman selanjutnya dari limit. (limit=4, page=2, menampilkan dari data ke-5)
    response
    Status Code

    User

    Pada bagian User diwajibkan mengirim Header x-auth-token dan Content-Type untuk setiap end point.

    headers
    x-auth-token *
    aweuaweu <token>
    Content-Type *
  • application/json
  • multipart/form-data

    • Content-Type bernilai 'application/json' jika tidak ada keterangan di penjelasan end point.

    Mendapatkan daftar user

    get
    /api/user

    Hanya bisa diakses oleh user yang memiliki role 'owner' dan 'super'

    query
    instansi
    <nama_instansi> (cth: Kemenko PMK)
    Mendapatakan user dari instansi terkait (hanya bisa oleh 'owner')
    role
    <nama_role> (cth: admin)
    Mendapatakan user berdasarkan role terkait (hanya bisa oleh 'owner')
    limit
    <angka> (cth: 10)
    Batasan total data yang akan diterima dan ditampilkan
    page
    <angka> (cth: 2)
    Menampilkan halaman selanjutnya dari limit. (limit=4, page=2, menampilkan dari data ke-5)
    response
    Status Code
    200 
                                        
                                        {
                                            "status": "success",
                                            "message": "Berhasil mendapatkan daftar user",
                                            "users": []
                                        }
